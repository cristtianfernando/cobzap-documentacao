# Escopo deste pacote

Este repositório entrega:

- arquitetura de informação inspirada em uma central de ajuda de software;
- textos originais adaptados ao posicionamento do Cobzap;
- estrutura compatível com GitBook e Git Sync;
- script para troca de marca em um export que você tenha autorização para utilizar;
- checklist de revisão e publicação.

Este pacote não reproduz integralmente textos, imagens, vídeos, códigos, políticas ou materiais proprietários de terceiros. Caso você possua um export autorizado, use o script incluído e faça a revisão técnica e jurídica antes da publicação.
