# API oficial e conexões alternativas

A API Oficial do WhatsApp é administrada pela Meta e utiliza uma conta WABA. Ela oferece governança empresarial, templates aprovados e regras formais de qualidade e envio.

Conexões alternativas podem apresentar comportamento, estabilidade e risco operacional diferentes.

Antes de escolher, avalie:

- criticidade do número;
- volume de mensagens;
- necessidade de templates;
- integração com sistemas;
- tolerância a indisponibilidade;
- políticas internas de compliance.

O Cobzap não elimina as regras, tarifas ou decisões de qualidade aplicadas pela Meta.
