# 🚀 Primeiro acesso

Siga esta sequência para colocar a operação em funcionamento:

1. confirme o acesso administrativo;
2. revise os dados da empresa;
3. cadastre usuários e equipes;
4. conecte um canal;
5. configure etiquetas e motivos;
6. importe uma base de teste;
7. realize atendimentos controlados;
8. valide dashboards e relatórios.
