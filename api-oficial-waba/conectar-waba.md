# Conectar uma conta WABA

1. Acesse **Configurações > Canais**.
2. Selecione **Adicionar canal**.
3. Escolha **WhatsApp API Oficial**.
4. Inicie a autenticação com a Meta.
5. Selecione o portfólio empresarial e a conta WABA.
6. Escolha ou cadastre o número.
7. Conclua as permissões.
8. Envie uma mensagem de teste.
9. Valide o recebimento na caixa de entrada.

> Os nomes dos menus podem variar conforme a versão da plataforma e o método de conexão habilitado.
