# 👋 Central de Ajuda Cobzap

Bem-vindo à Central de Ajuda Cobzap.

Este espaço reúne orientações para configurar a plataforma, conectar o WhatsApp, organizar equipes, acompanhar atendimentos, criar campanhas e utilizar automações na operação de cobrança.

## Encontre o que precisa

### Use a busca

Pesquise pelo nome da função ou pelo resultado desejado, por exemplo:

- conectar WhatsApp;
- cadastrar usuário;
- importar carteira;
- criar campanha;
- configurar template;
- acompanhar atendimento;
- consultar relatório.

### Navegue pelas categorias

O menu lateral está organizado conforme a jornada de implantação e uso:

1. conheça a plataforma;
2. prepare a conta;
3. configure administradores e equipes;
4. conecte os canais;
5. opere os atendimentos;
6. acompanhe indicadores;
7. automatize processos;
8. integre sistemas externos.

> **Importante:** menus e recursos podem variar conforme o plano contratado, o perfil de acesso e as integrações habilitadas.
