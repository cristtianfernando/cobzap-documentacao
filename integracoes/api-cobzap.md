# API Cobzap

A API permite consultar ou movimentar recursos autorizados da plataforma.

Antes de integrar:

1. obtenha a documentação da versão ativa;
2. gere credenciais exclusivas;
3. restrinja escopos;
4. armazene segredos com segurança;
5. trate paginação e limites;
6. registre erros sem expor dados sensíveis;
7. use ambiente de homologação quando disponível.
